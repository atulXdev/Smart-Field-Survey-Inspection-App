# React Native Mini Project Assignment

## Project: Smart Field Survey & Inspection App

### Objective
Build a complete React Native application using Expo APIs (Camera, Contacts, Location, Clipboard) with Expo Router, Bottom Tabs, Drawer Navigation, reusable components, and a modern UI.
